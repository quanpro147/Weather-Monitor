LIMIT 1 API-->>FE: 200 ApiResponse { data } FE->>API: GET /weather/{city_id}/advisory API->>RD: GET advisory:{city_id} Note right of RD: TTL = 3600s (1h) RD-->>API: hit/miss API->>SB: (nếu miss) latest record Note over API: Rule-based logic:
temp≥38°C → high risk
rain≥50mm → high risk
